Docker setup

Run Buddy Labels next to Bambuddy.

Buddy Labels only needs HTTP access to your Bambuddy API. It reads spool inventory, uses Bambuddy settings to build stable QR edit links, and serves the label editor from the container.

Docker Compose

Minimal service example

Use the released GHCR image, point BAMBUDDY_URL at your Bambuddy base URL, then open http://localhost:8787. 

services:
  buddy-labels:
    image: ghcr.io/ptheofan/buddy-labels:latest
    container_name: buddy-labels
    restart: unless-stopped
    ports:
      - "8787:8787"
    environment:
      BAMBUDDY_URL: "https://your-bambuddy.example.com"
      BAMBUDDY_API_KEY: "bb_your_bambuddy_api_key"
      APP_GITHUB_REPOSITORY: "ptheofan/buddy-labels"
      PORT: "8787"
Network note: use http://bambuddy:8000 if Bambuddy is another service on the same Compose network. Use http://host.docker.internal:8000 if Bambuddy runs directly on the Docker host.

Environment

Variables used by the container

BAMBUDDY_URL
Bambuddy base URL, without /api/v1. Example: https://buddy.example.com.
BAMBUDDY_API_KEY
Bambuddy API key. Buddy Labels sends it as both X-API-Key and bearer auth.
APP_GITHUB_REPOSITORY
Optional owner/repo value used by the browser version check. Default: ptheofan/buddy-labels.
PORT
Optional web server port inside the container. Default: 8787.
BAMBUDDY_BEARER_TOKEN
Advanced alternative to BAMBUDDY_API_KEY. Use one token variable, not both.

Bambuddy token

Recommended API key permissions

Grant the smallest set your Bambuddy version exposes. Buddy Labels does not create, update, delete, queue, or control anything in Bambuddy.

Required

  • inventory:read to read local or Spoolman inventory spools.
  • settings:read so Buddy Labels can read Bambuddy external_url for QR links.

If your UI uses broad checkboxes

  • Enable Manage Inventory only when there is no separate read-only inventory option.
  • Enable the settings/read-status option only if the app shows a Bambuddy connection warning.

Do not grant

  • Manage Queue, Control Printer, Manage Library, cloud access, or electricity price update.
  • Any write permission unless your Bambuddy build bundles read and write together.

QR links

Make Bambuddy edit links reachable

The exported QR code points to {external_url || BAMBUDDY_URL}/inventory?spool={spoolId}. If you scan labels from a phone or tablet, make sure Bambuddy's external URL is reachable from that device, or set BAMBUDDY_URL to a reachable URL.
