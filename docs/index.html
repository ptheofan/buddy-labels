Bambuddy label editor for EufyMake E1

Make your spool labels look the way you want.

Buddy Labels reads your Bambuddy spool inventory, lets you edit the label layout, and exports 75 x 53.5 mm SVG artwork for plastic spool label inserts. The QR code still resolves back to Bambuddy, and the SVG can outline text as paths for EufyMake Studio compatibility.

Workflow

How to use Buddy Labels

  1. Run the app. Open the hosted app in your browser, use ghcr.io/ptheofan/buddy-labels:latest, build the Docker image locally, or run pnpm run dev. Browser-hosted mode needs Bambuddy CORS access; Docker mode uses the built-in proxy.
  2. Import the template. Paste the share code into the app's Share code panel, click Import code, then save it as a named template.
  3. Select a spool. Search by color, material, spool ID, tag UID, tray UID, or tray UUID. The QR target should point to your Bambuddy /inventory?spool=... route.
  4. Export for EufyMake Studio. Keep Export text as paths enabled before exporting SVG. This avoids SVG text import issues in EufyMake Studio.
  5. UV print the insert. Import the SVG into EufyMake Studio, keep the artwork at 75 x 53.5 mm, and print onto the plastic insert face.
